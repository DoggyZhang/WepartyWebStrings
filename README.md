# WepartyWebStrings

设置转化模式
1. String转Excel
   输入以下内容
   mode=string2excel   string转excel模式
   newWeb=true|false   新/旧Web项目
   project=[path]      [path]为项目根路径
   output=[output]     [output]为结果输出路径,为空则默认为path

2. Excel转String
   输入以下内容
   mode=excel2string   excel转string模式
   newWeb=true|false   新/旧Web项目
   excel=[excel]       [excel]Excel翻译内容路径
   project=[path]      [path]为项目根路径
